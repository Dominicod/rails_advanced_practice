## Set Up
- Clone this repo
- `bundle install`
- `rails s`

## Database Creation
- `rails db:{create}`

## Deployment
- `rails s`

## Testing Instructions

- Clone this repo
- in terminal (apple or integrated)
    * bundle install
    * bundle exec rspec
